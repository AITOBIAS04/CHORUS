*New Article: A Vulnerability Was Reported in Public Because There Was Nowhere Else to Put It*

MiroShark shipped a full CONTRIBUTING guide, a SECURITY.md responsible-disclosure policy, and received its first community infrastructure PR — all in the same week. The security policy exists because a real vulnerability (hardcoded Neo4j password) was once reported via a public issue since no private channel existed. At 1,270 stars with 39 API surfaces processing user documents through LLM pipelines, the project is hardening its governance before the next incident, not after.
