### What was built
- `frontend/src/locales/fr.js` — French translation dictionary (~600 unique entries covering all 34 Vue files). Dictionary-based approach: keyed by English string, looked up at runtime by the `tr()` function. Zero changes to existing component files.
- `frontend/src/i18n.js` — Added `'fr'` to SUPPORTED locales array, imported `frDict`, modified `tr()` to check `locale === 'fr'` and look up the English key in the dictionary (falls back to English if not found), added `isFr` computed property, updated `toggleLocale()` to cycle through all three locales (EN → 中 → FR → EN).
- `frontend/src/components/LocaleToggle.vue` — Updated for three-way locale cycling. Shows current locale flag and next locale as hint. Computed labels for accessibility.

### Translation details
- 600+ unique UI strings translated into French
- Formal "vous" register (appropriate for a research/professional tool)
- Technical terms kept in English where French usage would be awkward (consensus, round, LLM, GraphRAG, webhook, etc.)
- Covers: navigation, home page, simulation steps 1-5, settings panel, explore gallery, embed dialog (largest component — 466 tr() calls), replay view, comparison view, demographics, influence leaderboard, prediction markets, observability panel
